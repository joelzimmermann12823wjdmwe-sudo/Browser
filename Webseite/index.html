Installations-Anleitung

1. Download starten

Klicke im Download-Center auf den blauen Windows-Knopf. Die Datei "Neon Browser Setup.exe" wird heruntergeladen.

2. Windows-Sicherheit (Wichtig!)

Da NEON ein neues Projekt ist, zeigt Windows evtl. "Der Computer wurde durch Windows geschützt". Klicke auf "Weitere Informationen" und dann auf "Trotzdem ausführen".

3. Die Installation

Der Installer erledigt den Rest von selbst. Sobald der Ladebalken fertig ist, öffnet sich dein neuer Browser automatisch.

4. Fehlerbehebung

Sollte es Probleme geben, stelle sicher, dass du alte Versionen über den Taskmanager (Strg+Shift+Esc) geschlossen hast.
